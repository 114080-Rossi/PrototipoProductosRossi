Descripción del Proyecto
Este proyecto es una aplicación web personal de gestión para mi emprendimiento de panadería. Su objetivo principal es facilitar el control de ventas, clientes, proveedores, productos y materia prima. La plataforma permite generar facturas, registrar compras e ingresos de materia prima, gestionar el stock y llevar el seguimiento de los saldos de cada cliente o proveedor.

La solución está desarrollada con:

Backend: Java con Spring Boot y Maven

Base de datos: SQL Server 2021

Frontend: HTML, JavaScript y Angular

Modelo de datos: Totalmente normalizado, con control de stock, historial de movimientos y gestión de cuentas corrientes

El sistema incluye tablas para Clientes, Proveedores, Direcciones, Productos, Categorías, Materias Primas, Facturas, Detalles de Factura, Compras, y Movimientos de Cuenta. Próximamente se integrará lógica automática para actualizar saldos en cada operación de venta o compra.
